# Full WFC Lab

This resettable lab proves the continuous-facility simulation before it is assembled
into the played game. It visualizes all three levels at once, advances deterministic
decoherence pulses, and makes the observation boundary explicit.

## Question

Can unobserved rooms and halls be regenerated by constrained WFC while the current
player, observed threshold chain, and one claimed exit route remain stable and every
player retains an A* route to the exit?

## Controls

- `Arrow keys`: move the simulated player across a horizontal connection.
- `PageUp` / `PageDown`: traverse an open vertical connection.
- `1`–`6`: look through east, west, south, north, up, or down threshold.
- `0`: stop observing a threshold.
- `Space`: force one decoherence pulse.
- `P`: pause/resume timed pulses.
- `R`: reset with the next seed.

## Visual legend

- Cyan diamond: local player (`PlayerId(0)`).
- Green diamond: exit.
- Gold tiles: current A* route used by the candle.
- Purple outline: geometry pinned by current observation/occupation.
- Magenta threshold bar: the currently observed threshold.
- Dark blue: room; slate: hall; near-black: void.

Success means every accepted pulse changes only unobserved geometry, preserves the
player-to-exit route, and never opens a second threshold into an already claimed exit.
Failure is displayed in the top-left status panel and leaves the last valid generation
intact.
