# TrenchBroom Editor Configuration Tool

This folder contains the TrenchBroom game configuration package and installation script for Observed 2.

---

## Files

- **`install_config.ps1`**: PowerShell script to install `Observed2/GameConfig.cfg` and `Observed2/Observed2.fgd` into TrenchBroom's appdata games directory.
- **`Observed2/`**: Contains the Forge GameData (`.fgd`) entity definition file and TrenchBroom game configuration file.

---

## Installation

Run PowerShell:
```powershell
.\tools\trenchbroom\install_config.ps1
```
